# DAY4 自动化备份脚本

## 简介
这是一个用于 Linux 环境下的自动化备份脚本，功能是遍历指定目录下的所有子目录，并按当前日期打包成压缩文件，存放到指定输出目录。  

适合日常备份文件夹、实验数据或者代码目录，方便管理和归档。

---

## 功能特点
- 自动遍历源目录下所有子目录。
- 按子目录分别生成 `.tar.gz` 压缩包。
- 压缩包命名格式：`backup_YYYYMMDD_目录名.tar.gz`。
- 自动创建输出目录（如果不存在）。
- 打印备份成功或失败的日志，并带时间戳。
